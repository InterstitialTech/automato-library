# Automato Sensor Board Library 

This library implements the Automato API via the "Automato" class interface.


## Dependencies
* Adafruit GFX
* Adafruit ILI9341
* SparkFun SHTC3

## Manual Installation
In your Arduino directory:
```
git clone https://github.com/InterstitialTech/automato-library.git
```
